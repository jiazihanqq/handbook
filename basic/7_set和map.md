# map和set
## set（集合）
### 特性
- 是一个类似数组的对象
- 成员唯一且无序
- Array.from 转换为数组
### 基本方法
- add
- delete
- has
- clear
### 遍历方法
- keys
- values
- entries
- foreach

## weakset
- 存放对象
- 如果不使用，自动回收
- 不可遍历
### 基本方法
- add
- has
- delete

## Map（字典）
- 集合 是以 [value, value]的形式储存元素，字典 是以 [key, value] 的形式储存
### 基本方法
- get
- set
- has
- delete
- clear
### 遍历方法
- keys
- values
- entries
- foreach
